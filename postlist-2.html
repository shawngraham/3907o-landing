Setting up a a professional page on Github

There are many different options for setting up a home base for your 'online identity'. I created a repository in Github called shawngraham.github.io. Everything in that repository gets served up as a proper website whenever someone goes to http://shawngraham.github.io. In the video below, Alex Galarza takes you through setting up such a space, using a Twitter Bootstrap pre-cooked theme.

You can set a blog up in that same space; here's something called 'Jekyll-Now' which you can fork into your .github.io repository.
